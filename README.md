# AutoAnswer-
一款基于 AI 的桌面答题神器。框选屏幕区域，程序自动截图、识别题目文字、解析选项，秒级调用大模型推理出答案，结果直接显示在半透明悬浮窗上——全程无需手动操作。  内置 50+ 条本地知识库，命中即返回，响应低至毫秒级；未命中则调用 DeepSeek API 联网搜索作答，准确率极高。OCR 引擎采用 RapidOCR，无需安装庞大的深度学习框架，开箱即用。  支持全局快捷键一键框选、暂停、手动触发识别，操作丝滑不打断工作流。悬浮窗置顶显示、自动淡出，答案一目了然。  适用于线上知识竞赛、时政学习、模拟测验、趣味问答等场景。轻量、快速、安静，藏在后台帮你拿分的那个队友。
